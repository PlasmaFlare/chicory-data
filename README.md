## Development

Requirements:

- [Node v14+](https://nodejs.org)
- [Yarn](https://yarnpkg.com)

Steps:

1. git clone
2. yarn
3. yarn start

This should open the app at http://localhost:3000

This project was bootstrapped with [Create React App](https://create-react-app.dev/docs/getting-started).
